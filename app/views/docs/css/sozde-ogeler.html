CSS Sözde Öğeler - Pseudo Elements

CSS sözde öğeleri, bazı seçicilere özel efektler eklemek için kullanılır.

Sözde Öğeler Yazım Kuralı

    
secici:sozde-oge {
    ozellik: deger;
}

Örnek olarak görelim.

    
p:first-line {
    color: orange;
}
    

    
.ozel-stil:first-letter {
    color: blue;
}

:first-line Özelliği

Bir elemenanın ilk satırına stil vermek için kullanılır. Sonra ki satırlar normal olarak görüntülenir.

    
p:first-line {
    color: orange;
}

:first-letter Özelliği

Bir elemenanın ilk satırında ki ilk harfine/karakterine stil vermek için kullanılır. Sonra ki harfler/karakterler normal olarak görüntülenir.

    
p:first-letter {
    color: orange;
    font-size: 25px;
}

:after ve :before Özellikleri

Bir elemanın önüne CSS ile bir içerik eklenmek istendiğinde ":before" sözde öğesi kullanılabilir. Bir elemanın arkasına ise CSS ile bir içerik eklenmek istendiğinde ":after" sözde öğesi kullanılabilir.

CSS Kodumuz:

    
p:before {
    content: "Merhaba!";
}

h1:after {
    content: url(dosya1.jpg);
}

Bu kod çalıştırıldığında her "p" etiketinin başına "Merhaba!" yazacak ve her "h1" etiketinin sonuna "dosya1.jpg" adlı gorseli ekleyecek.
