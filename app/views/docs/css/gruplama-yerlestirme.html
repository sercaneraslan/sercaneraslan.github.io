CSS Seçicileri Gruplama - Grouping Selectors

Aşağıdaki gibi bir kodumuz varsa;

    
h1 {
    color: #DBDBDB;
}

h2 {
    color: #DBDBDB;
}

p {
    color: #DBDBDB;
}

Yani tüm seçicilerin özellikleri ve değerleri aynı ise bu seçicileri virgül ile ayırarak gruplayabiliriz. Ayrıca bu durumda kod tasarrufu yapıp daha az kod yazmış olacağız. Yukarıda ki kodu nasıl küçültüp grupladığımızı görelim.

    
h1,
h2,
p {
    color: #DBDBDB;
}

CSS Seçicileri İç içe Yerleştirme - Nesting Selectors

Aşağıdaki gibi bir XHTML ve CSS kodumuz var;

    
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http: //www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http: //www.w3.org/1999/xhtml" lang="en" xml: lang="en">
    <head>
        <meta http-equiv="content-type" content="text/html; charset=utf-8;" />
        <link rel="stylesheet" type="text/css" href="dosyaAdi.css" />
        <title>Başlık Buraya</title>
    </head>
    <body>
        <div class="ozel-makale">
            <p>Bu bir paragraf.</p>
        </div>
        <p>Bu da bir paragraf.</p>
    </body>
</html>
    

    
.ozel-makale {
    background-color: blue;
}

p {
    color: gray;
}

Yukarıdaki CSS kodumuz tüm "p" etiketlerinin içerik renklerini "gri" olarak, ve "ozel-makale" class'lı div'imizin arkaplan rengini "mavi" olarak tanımladı.

Biz "ozel-makale" class'lı div'in altında ki "p" etiketinin içerik renginin "beyaz" olmasını istersek şu kullanımı uygulayacağız.

    
.ozel-makale p {
    color: white;
}
