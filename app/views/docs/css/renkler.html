CSS Renkler - Colors

Renkler, Kırmızı (Red), Yeşil (Green) ve Mavi (Blue) “Işık” renklerinin kombinasyonu ile oluşturulur (RGB). Işık renklerinde ana renkler Kırmızı, Yeşil ve Mavi'dir. HTML/XHTML renkleri, Hexadecimal değerlerle ve RGB kombinasyonuyla belirtilebilir. En düşük değer 0 (Hex #00) ve en yüksek değer de 255′tir (Hex #FF).

CSS Güvenli Web Renkleri - Web Safe Colors

Bilgisayarların sadece 256 rengi desteklediği zamanlarda 216 adet renk, Web standartı olarak önerildi. Örnek kullanımı görelim.

    
.paragraf-stilim {
    color: #3366CC;
}
