CSS Seçiciler - Id & Class Selector

Seçiciler, Id ve Class olmak üzere ikiye ayrılır.

CSS Tekil Seçici - Id Selector

Daha önce HTML/XHTML etiketlerine stil vermeyi görmüştük. Örneğin "p" etiketine stil verdiğimizde tüm "p" etiketlerinde bu stilin olacağını biliyoruz. Eğer biz sadece tek bir "p" etiketinde stilimiz gözüksün istiyorsak "Id" seçicisini kullanacağız. "Id" seçicisi, bir kez kullanılabilir. Örneğin sadece tek bir paragrafa stil vereceksek ve bu stili başka hiçbir yerde kullanmayacaksak "p" etiketi içine id="isim" yazacağız. "isim" olarak belirttiğimiz yere istediğimiz isimi yazabiliriz, kodumuzu daha sonra bu isim ile CSS kodunda çağıracağız. (İsim, rakamla başlamamalı ve Türkçe karakter içermemelidir.)

Dosyalama Mantığı

HTML/XHTML kodumuz ile CSS kodumuzu aynı klasör içinde tutmalıyız. Örneğin, "test.com" diye bir sayfa yazıyoruz. "test.com" adı ile bir klasör oluşturmalıyız ve tüm dosyalarımızı bu klasör içinde tutmalıyız. Görsellerimizi "img" ya da "images" klasörü içinde, CSS dosyalarımızı "css" ya da "styles" klasörü içinde ve HTML sayfalarını da "views" klasörü içinde tutabiliriz. Böylelikle herşey daha sistemli bir hale gelecektir.

Klasör Çağırma Mantığı

Örneğin bizim CSS dosyamız "test.com/styles" klasörümüzün içinde "style.css" adı ile duruyor. HTML/XHTML dosyamızda "test.com/index.html" olarak duruyor. Bu durumda HTML/XHTML kodumuzda "link" etiketi içinde bu dosyayı "styles/style.css" olarak çağıracağız. Eğer HTML/XHTML sayfamız "test.com/views/index.html" olarak dursaydı "../styles/style.css" olarak çağırmamız gerekecekti. ("../" karakterleri "Bir üst klasöre çık" anlamına gelmektedir.)

HTML/XHTML kodumuzu nasıl yazdık görelim...

    
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
    <head>
        <meta http-equiv="content-type" content="text/html; charset=utf-8;" />
        <link rel="stylesheet" type="text/css" href="styles/style.css" />
        <title>Başlık Buraya</title>
    </head>
    <body>
        <p id="paragraf-stilim">İçerik Buraya</p>
    </body>
</html>

Şimdi HTML/XHTML etiketine verdiğimiz "Id" yi CSS kodunda çağıralım. CSS kodumuzu yazarken "#" ile başlayıp boşluk vermeden "Id" ismini yazıyoruz ( #paragraf-stilim ). Sonrasında ise CSS yazım kuralını uyguluyoruz. "color: blue;" renk mavi olsun anlamına gelmektedir. Renkleri bir sonra ki konuda ele alacağız.

    
#paragraf-stilim {
    color: blue;
}

CSS Çoğul Seçici - Class Selector

"Class" seçicisi birden fazla kez kullanılabilir. Örneğin, yazı rengi mavi olsun istediğimiz tüm etiketlere sayısız kere "Class" seçicisi yazabiliriz. "Id" seçicisinde "p" etiketi içine id="isim" şeklinde yazıyorduk, "Class" seçicisinde ise "id" yerine "class" yazacağız. Örneğin: class="isim"

HTML/XHTML kodumuz nasıl yazılmalı görelim...

    
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
<html xmlns="http://www.w3.org/1999/xhtml" lang="en" xml:lang="en">
    <head>
        <meta http-equiv="content-type" content="text/html; charset=utf-8;" />
        <link rel="stylesheet" type="text/css" href="styles/style.css" />
        <title>Başlık Buraya</title>
    </head>
    <body>
        <p class="paragraf-stilim">Bu içerik mavi renk.</p>
        <span class="paragraf-stilim">Bu içerikte mavi renk.</span>
    </body>
</html>

"Id" seçicisinin CSS kodunu yazarken "#" karakterini kullanıyorduk. "Class" seçicisinde ise "." (nokta) karakterini kullanacağız.

    
.paragraf-stilim {
    color: blue;
}
