CSS Giriş

CSS öğrenebilmemiz için HTML/XHTML bilmeniz gerekiyor. Eğer bilmiyorsanız XHTML konusuna bakabilirsiniz.

CSS nedir?

CSS'in açlımı "Cascading Style Sheets"tir. Türkçesi ise "Stil Şablonları" olarak geçmektedir. HTML/XHTML öğelerine stil vermek için kullanılır. Bu dili yazabilmek için metin editörlerini (Not Defteri, TextMate, Sublime Text, Vim, Notepad++ vs) kullanabilirsiniz.

CSS, HTML/XHTML ile stilleri birbirinden ayırmak için World Wide Web Consortium (W3C)'u tarafından yaratılmıştır.

CSS ile sayfamızın tamamına stil verebileceğimiz gibi sadece tek bir harfe de stil verebiliriz, CSS burada bize çok büyük bir esneklik gösteriyor.

CSS ile Neler Yapılabilir?

CSS kullanarak HTML/XHTML ile yapamayacağımız birçok şeyi yapabiliriz. Bir sayfada ki tüm öğeleri veya belli bir öğeyi renk, boyut, arkaplan görseli, sayfada bulunan konumu, bir öğenin diğer öğeler ile arasında ki mesafeyi ve daha bir çok özelliği CSS ile tanımlayabiliriz.

Internet Explorer Faktörü

Internet Explorer ve diğer gelişmiş tarayıcılar (Firefox, Chrome, Safari, Opera vs.) farklı yorumlar ortaya koyarlar. Bu durumda tüm tarayıcılarda aynı/benzer yorumlanacak kodlar yazmak en iyi seçim olacaktır.
