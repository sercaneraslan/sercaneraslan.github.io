HTML5 Application Cache - Uygulama Cache'i

Manifest dosyası oluşturarak kodlarımızı/dosyalarımızı cache'letebiliriz/zulalatabiliriz. Tabi bu durumda kullanıcıların çevrimdışı/offline durumlarda da (internet bağlantısı olmaksızın) dosyaları kullanabilmesini sağlamış oluruz. Dosyaların cache'lenmesi kullanıcılara çok iyi bir hız deneyimi yaşatır. Server tarafında ki yük ise fazlasıyla düşecektir çünkü kullanıcı sadece güncellenen/eklenen verileri çekecektir.

Gelişmiş tarayıcıların hepsi "Application Cache" özelliğini destekler.

HTML5 Cache Manifest Örneği

    
<!DOCTYPE html>
<html manifest="demo.appcache">
    <body>
        Döküman içeriği...
    </body>
</html>

Cache Manifest Basics - Temel Manifest Cache'i

Dökümandaki html elemanı içine manifest özniteliği ekleyerek Application Cache'i kullanabilir duruma getirebiliriz.

    
<html manifest="demo.appcache">

Örnek bir Manifest Dosyası içeriği

    
CACHE MANIFEST
index.html
stylesheet.css

scripts/main.js
