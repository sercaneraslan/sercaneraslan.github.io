CSS Öznitelik Seçicileri - Attribute Selectors

HTML/XHTML elemanlarımızın içine yazdığımız öznitelikleri (title, type vs) seçici olarak kullanıp elemanlarımıza stil verebiliriz. Öznitelikleri CSS'te "[" ve "]" işaretleri arasında yazacağız.

    
[title] {
    color: orange; 
}

Yukarıda ki kod, içinde "title" özniteliği bulunan bütün elemanların metin rengi turuncu yapacaktır.

Öznitelik ve Değer Seçicisi

Özniteliği ve değerini birlikte kullanarak elemanlara stil verebiliriz.

    
[title=SercanEraslan] {
    background-color: yellow;
}

Yukarıda ki kod, sadece "title" değeri "SercanEraslan" olan elemanların arkaplan rengini sarı yapacak.

    
[title~=bilgi] {
    color: green;
}

Yukarıda ki kod, "title" değerlerinde "bilgi" geçen bütün elemanların metin rengini yeşil yapacak.

Form Özniteliklerine Stil Verme

    
input[type="button"] {
    height: 50px;
}

Yukarıda ki kod, "type"ı "button" olan bütün input'ların yüksekliğini 50 piksel yapacaktır.
