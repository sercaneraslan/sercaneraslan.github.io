HTML5 Yeni Input Tipleri

HTML5 formları için birçok yeni input tipleri gelmiştir. Bu yeni özellikler, iyi bir input kontrolü ve doğrulaması sağlar.

Not: Tüm gelişmiş tarayıcılar yeni input tiplerini destekler. Ancak tarayıcılar yeni input tiplerini desteklemiyorsa bunlar normal text alanları gibi gözükecektir.

Yeni input tipleri:

color (renkler için)

    
<input type="color" name="favoriRenk" />

date (doğumgünü, yıldönümü vs için)

    
<input type="date" name="dogumgunu" />

datetime (date tipine ek olarak saati de kapsar)

    
<input type="datetime" name="dogumgunuSaat" />

datetime-local (datetime tipine ek olarak zaman dilimi tanımlar)

    
<input type="datetime-local" name="dogumgunuSaatZamanDilimi" />

email (e-mailler için)

    
<input type="email" name="kullaniciEmail" />

month (ay bilgisini yazmak için)

    
<input type="month" name="Ay" />

number (sadece rakam yazmak için. min, max, step ve value öznitelikleri tanımlanabilir.)

    
<input type="number" name="adet" min="1" max="5" />

range (Fiyat aralığı vb yazmak için. min, max, step ve value öznitelikleri tanımlanabilir.)

    
<input type="range" name="fiyat" min="1" max="10" />

search (arama inputları için.)

    
<input type="search" name="arama" />

tel (telefon yazmak için)

    
<input type="tel" name="kullaniciTelefonu" />

time (Saat için)

    
<input type="time" name="saat" />

url (url girebilmek için)

    
<input type="url" name="anasayfa" />

week (hafta bilgisini yazmak için)

    
<input type="week" name="hafta" />
