HTML5 Canvas

<canvas> elemanı web sayfasına grafik çizmek için genellikle JavaScript ile kullanılır. <canvas> "height" ve "width" öznitelikleri ile çizilebilir bir alan oluşturur. Canvas ile kutular, daireler, karakterler, görseller ve daha fazlasını çizdirebilirsiniz.

<canvas> elemanını tanımlarken "id", "width" ve "height" öznitelikleri verilmelidir.

    
<canvas id="myCanvas" width="200" height="100"></canvas>

JavaScript ile Çizim

<canvas> elemanının kendi çizim yetenekleri yoktur. Tüm çizimler JavaScript içinde yapılır.

    
<script type="text/javascript">
    var myCanvas = document.getElementById('myCanvas'),
        ctx = myCanvas.getContext('2d');

        ctx.fillStyle = '#2cc185';
        ctx.fillRect(0, 0, 150, 75);
</script>

JavaScript, id kullanarak <canvas> elemanını bulur.

    
var myCanvas = document.getElementById('myCanvas'),

Sonra bir içerik nesnesi oluşturur. GetContext("2d") nesnesi yerleşik bir HTML5 nesnesidir.

    
ctx = myCanvas.getContext('2d');

Aşağıdaki iki satır, kırmızı bir dikdörtgen çizer:

    
ctx.fillStyle = '#2cc185';
ctx.fillRect(0, 0, 150, 75);

fillStyle özniteliği rengi yeşil yapar, fillRect ise şeklini, konumunu ve boyutunu belirler. ilk iki değer yukarıda ve soldan olan mesafeyi ayarlar son iki değer ise oluşturduğumuz kutunun genişliğini ve yüksekliğini belirler.
