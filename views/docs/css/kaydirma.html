CSS Kaydırma - Float

Blok elemanlarımızı "float" özelliği sağa veya sola yaslayabiliriz. Kaydırmalar tam sol ve tam sağ olarak yapılır. Örneğin bir elemanı sola yasladığımızda o mümkün olduğunca en sola yaslanacaktır.

"left" ve "right" olmak üzere iki değeri mevcuttur.

"float" özelliğini daha iyi anlamak için aşağıdaki örneği inceleyiniz.

CSS Clear Özelliği - Clear Property

"float" uygulanan elemandan sonra ki yazılan içeriklerde/kodlarda kaymalar yaşanabilmektedir. Bu sebeple "float" kullanımından sonra clear: both; özelliği kullanılması gerekebilir. Genelde yapı düzgün tasarlandığında bu özelliğe ihtiyaç olmuyor. :before veya :after özellikleri ile kullanılması önerilir.

Örnek olarak görelim.

    
.float-temizle {
    clear: both;
}
