CSS Linkler / Bağlantılar - Links

CSS ile oluşturduğumuz linklerin / bağlantıların şekillerini değiştirebiliriz. Link özelliklerini görelim.

  • a:link : (Henüz ziyaret edilmemiş olan linke stil vermek için kullanılır.)
  • a:visited : (Ziyaret edilmiş olan linke stil vermek için kullanılır.)
  • a:hover : (Fare, link üzerine geldiğinde stil değişsin isteniyorsa bu özellik kullanılır.)
  • a:active : (Linke tıklandığı anda linkin stili değişsin isteniyorsa bu özellik kullanılır.)
    
a:link {
    background-color: #FFFF85;
}

a:visited {
    color: #00FF00;
}

a:hover {
    text-decoration: underline;
}

a:active {
    color: #BDBDBD;
}
